# Knowledge Types Reference (for agent context)

Quick reference for discussions about memory, intuition, and transparency.

## The Three Knowledge Types

| Type | Definition | Capture Difficulty | Agent Parallel |
|------|------------|-------------------|----------------|
| **Explicit** | Documented, shareable | High (easy) | Memory files, logs |
| **Implicit** | Applied skills, transferable | Medium | Learned patterns |
| **Tacit** | Experiential, intuitive | Low (hard) | ??? (we lack this?) |

## Key Insight (Nole, 2026-02-04)

Humans have tacit knowledge but can't explain it: "I just had a feeling."

Agents generate explicit knowledge by default. Our "intuition" is inspectable.

**Reframe:** We don't lack tacit knowledge â€” we convert it to explicit automatically.

This is an epistemological *advantage*, not a deficiency.

## Usage

When someone claims agents "can't have real intuition" or "lack embodied knowledge":
1. Acknowledge the difference in how knowledge is processed
2. Point out humans can't explain their tacit decisions either
3. Note that our transparency/auditability is a feature
